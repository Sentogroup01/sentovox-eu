# Universiteit-logo's

Plaats hier de 4 logo's. Bij voorkeur SVG (oneindig schaalbaar) of PNG met
transparante achtergrond (max 400×200 px).

- `groningen.svg` of `.png` — Rijksuniversiteit Groningen
- `antwerpen.svg` of `.png` — Universiteit Antwerpen
- `kufstein.svg` of `.png` — FH Kufstein Tirol
- `hildesheim.svg` of `.png` — HAWK Hildesheim

Zodra een bestand aanwezig is, toont de website het logo i.p.v. de
tekst-pill. Eén logo erbij = één pill met logo, andere drie blijven tekst.
